The tesseract app is very important to run the application without crashing.
The machine learning models cannot read the pixels directly from the jpg and jpeg pictures. the tesseract is used and functioned as a translator that converts visual characters into the digital texts that the python code can understand and process. 
the tesseract uses the " pytesseract " model that is used to extract the text out of an image or screenshot. this extracted text then passed over to the models to analyze and predict the outcomes. 
for more accuracy we also implemented the " OpenCV " module , that converts the image and screenshot into grayscale and apply thresholding ( making the image totally black and white ) before giving to the tesseract.
we do this process because the OpenCV is used to remove the background noise in the image and by doing this tesseract can act more efficiently by predicting maximum correct outputs.

How tesseract implemented in our app:
 the tesseract is integrated into the " extract_text_from_image " function in our app:
1- INPUT : uploading image or screenshot of the job description.
2- PROCESSING : it identifies the characters.
3- OUTPUT: The extracted text is added to the combined_text variable.
4- FINAL STEP: This text is sent to your predict() function to check for fraud.

if sir asks why this model .this is the answer:
 A standard text model is limited to copy-pasted text. By integrating Tesseract OCR, I have expanded the system's capability to detect 'Visual Scams.' This allows the tool to analyze screenshots from social media and messaging apps, which are the primary channels where modern job fraud occurs.
